# Stoikaya Beauty — сайт записи клиентов

Небольшой сайт для мастера бьюти-услуг с возможностью просмотра свободных окон и ручной записи клиентов.

Проект сделан как простой и понятный MVP без серверной части.

## Для кого
Сайт предназначен для личного использования мастером:
- просмотр свободных и занятых слотов
- ведение записей с одного устройства
- быстрый доступ для клиентов через соцсети

## Возможности
- календарь с выбором месяца и дня
- список временных слотов
- бронирование и снятие брони кликом
- сохранение записей в `localStorage`
- фильтр «Показать занятые»
- адаптация под мобильные устройства

## Как открыть сайт
Он доступен по ссылке (GitHub Pages):

https://fizalion.github.io/stoikaya-beauty/

Или локально:
1. Склонировать репозиторий
2. Открыть `index.html` в браузере

## Технологии
- HTML
- CSS
- Vanilla JavaScript
- LocalStorage
